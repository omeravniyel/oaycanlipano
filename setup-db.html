⚠️ Veritabanı Kurulumu

Yeni çoklu kurum sistemine geçmek için Supabase veritabanında yeni bir tablo oluşturulması gerekiyor. Aşağıdaki butona basarak kurulum talimatlarını izleyin.

Supabase SQL Editor'de çalıştırılacak komut:

CREATE TABLE institutions (
  slug text PRIMARY KEY,
  name text,
  password text,
  config jsonb DEFAULT '{}'::jsonb
);

-- Önceki verileri kaybetmemek için varsayılan bir kurum oluştur:
INSERT INTO institutions (slug, name, password, config)
VALUES ('omeravniyel', 'Ömer Avni Yel Öğrenci Yurdu', '1234', '{}');

Nasıl Yapılır?

  1. Supabase Paneline gidin.
  2. Soldaki menüden SQL Editor'e tıklayın.
  3. Yukarıdaki kodu kopyalayıp yapıştırın.
  4. RUN butonuna basın.
  5. İşlem tamamlandığında bu sayfayı kapatabilirsiniz.
